1. Cvienie

Pre systm v tvare $\dot{x}(t)=ax(t)+bu(t)$

Kde $b = 1$ a $a$ je neznme

  • Systm je prvho rdu
  • Prenosov funkcia je $S(s)=\frac{1}{s-a}$
  • Charakteristick polynm: $s-a$
  • Charakteristick rovnica: $s-a=0$
  • Systm je stabiln pre $a < 0$ a nestabiln pre $a > 0$
  • Zosilnenie je 1 a asov kontanta je 1
  • Prkladom me by dolnopriepustn filter (elektronick)

def sim_fn1 (x,t,u,a,b):
    return a*x + u*b

def sim_it(sim_fn, sim_start, sim_end, sim_step, *args):
    time_vec = np.arange(sim_start,sim_end + sim_step, sim_step).reshape(-1,1)

    sim_out = odeint(sim_fn,args[0],time_vec[:,0], args=args[1:])
    return [time_vec, sim_out]
    
def printPlot(x_val, y_val, title:str = "", xLabel:str = "", yLabel:str = ""):
    fig = plt.figure()
    ax = fig.add_axes([0,0,1,1])
    ax.plot(x_val, y_val)
    ax.axhline(y=0,color='r',linestyle='-')
    ax.set_title(title)
    ax.set_ylabel(yLabel)
    ax.set_xlabel(xLabel)
    ax.grid()

    x,k = x_
